Q5.Demonstrate how to deploy “factorial of a number” java application using a Dockerfile? 

Step:
1.cd Downloads
2.mkdir docker-java-app
3.cd docker-java-app
4.nano Factorial.java
class Factorial {
    public static void main(String[] args) {
        int num = 10;
        long factorial = 1;
        for(int i = 1; i <= num; ++i)
        {
            factorial *= i;
        }
        System.out.printf("Factorial of %d = %d", num, factorial);
    }
}

5.nano Dockerfile
From openjdk
LABEL author='apsit'
COPY . /var/www/java
WORKDIR /var/www/java
RUN javac Factorial.java
CMD ["java","Factorial"]

6.docker build -t apsit/javaapp .
7.docker images
8.docker run -d --name apsit_javacontainer [idno]
9.docker ps -a
10.docker run -d --name apsit_javacontainer1i9 [idno] tail -f /dev/null
11.docker ps -a 
12.docker exec -it [2nd dockerconatinerid] bash
13.javac Factorial.java
14.java Factorial
