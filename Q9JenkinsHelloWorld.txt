from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.common.keys import Keys
import time
# Initialize Chrome driver
driver = webdriver.Chrome()
# Step 1: Open form page
driver.get("https://www.w3schools.com/html/html_forms.asp")
driver.maximize_window()
# Step 2: Find input fields by name
fname = driver.find_element(By.ID, "fname")
Subject Incharge : Sujata Oak Department of Information Technology
lname = driver.find_element(By.ID, "lname")
# Step 3: Clear and fill the form fields
fname.clear()
fname.send_keys("Sujata")
lname.clear()
lname.send_keys("Oak")
# Step 4: Scroll and click Submit button
driver.find_element(By.XPATH, "//input[@type='submit']").click()
# Step 5: Wait and print the new URL
time.sleep(5)
print("Form submitted successfully!")
print("Redirected URL:", driver.current_url)
# Step 6: Close browser
driver.quit()

18.Create a New Job
On the Jenkins dashboard, click "New Item".
Enter an item name: HelloWorld-Job.
Choose "Freestyle project" and click "OK".
On the configuration screen, scroll to the "Source Code Management" section.
Select "Git".
In the "Repository URL" field, paste the URL of your Git repository containing the java file.
https://github.com/USERNAME/Simple-HelloWorld-Java.git

19.Add Build Steps:
Scroll to the "Build Steps" section.
Click "Add build step" and select "Execute shell".
In the Command text area, enter the following commands:
javac HelloWorld.java
java HelloWorld
Click "Save".

20.Run the Build:
In the left-hand menu, click "Build Now".
In the "Build History" panel on the left, click on the latest build number 
Click on "Console Output".
Scroll to the bottom of the log to confirm the successful execution and see your program's output.
